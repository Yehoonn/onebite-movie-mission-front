# 🎬 Onebite Cinema

한입 챌린지 8기 **Next.js 2주차 프로젝트**  
영화 데이터를 조회하고 검색할 수 있는 **영화 정보 웹 서비스**입니다.

프론트엔드는 **Next.js**로 구현되었으며, 백엔드 서버와 연동하여 실제 데이터를 기반으로 동작합니다.

## 🚀 배포 링크 (Vercel)

https://onebite-movie-mission-front-git-main-kyhs-projects.vercel.app/

## 📌 주요 기능

### 1. Home (/)

- 백엔드 서버에서 영화 데이터를 불러와 렌더링
- **추천 영화 섹션**
  - 랜덤 영화 3개 표시
  - **3초마다 자동 변경**
- **전체 영화 목록**
  - 데이터베이스의 모든 영화 출력
- 페이지 캐싱 적용

### 2. Search (/search?q=검색어)

- 영화 제목 검색 기능
- 검색 결과 **CSR 방식 데이터 페칭**
- 검색어 기반 영화 리스트 렌더링

### 3. Movie Detail (/movie/[id])

- 영화 상세 정보 조회
- 백엔드 API로부터 데이터 요청
- **모든 영화 페이지 캐싱 적용**

### 4. SEO 설정

- favicon 설정
- Open Graph 메타 태그 적용
- 썸네일 이미지 설정
